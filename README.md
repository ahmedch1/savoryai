# SavoryAI

## Project Overview
Interactive food search and RAG (Retrieval-Augmented Generation) chatbot system with an intuitive web interface for user interaction.

## Project Goals
- Build an AI-powered food search and recommendation system
- Implement RAG chatbot for food-related queries
- Deploy the system online for public access
- Create an intuitive interface for seamless user interaction

## Architecture
**Backend (Python):**
- RAG (Retrieval-Augmented Generation) system
- Food search engine
- API endpoints
- Vector database integration
- LLM integration

**Frontend:**
- Interactive web interface
- Chat interface
- Search functionality
- User-friendly design

## Tech Stack
**Backend:**
- Python 3.x
- FastAPI / Flask (API framework)
- LangChain / LlamaIndex (RAG framework)
- ChromaDB / Pinecone / Weaviate (Vector database)
- OpenAI / Anthropic / Local LLM

**Frontend:**
- React / Next.js
- TypeScript
- Tailwind CSS

**Deployment:**
- Docker
- Cloud platform (AWS / GCP / Azure / Vercel)
